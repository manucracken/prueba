CSS

La idea de CSS es la de utilizar el concepto de seaparación de presentación y contenido. CSS son las siglas en inglés para <> (Cascading Styles Sheets). Básicamente, es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita

Ventajas de usar CSS

  • Separación de le estructura y la presentación. CSS permite separar el contenido HTML de su presentación visual. Es decir, te permite mantener el código HTML, limpio y estructurado, mientras que el estilo se define en un archivo CSS separado. Este ordfen mejora la legibilidad del código y facilita el mantenimiento, así como la actualización de los estilos.

  • Consistencia y mantenibilidad. Al utilizar CSS, puedes aplicar estilos de manera consistente a un sitio o aplicación web. Los estilos se definen una vez y se aplican a múltiples elementos en las páginas, lo que favorece la renovación de la apariencia visual de todo el proyecto
